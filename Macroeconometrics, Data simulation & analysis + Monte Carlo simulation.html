Atsižvelgiant į nurodytas duomenų generavimo sąlygas, tai yra dinaminis panelinis modelis. Tą galima pastebėti iš to, jog yra įtrauktas priklausomas 'lag' kintamasis $Y_{it - 1}$ kaip vienas iš prediktorių. Pastarasis narys atspindi priklausomo kintamojo $Y$ reikšmę tam pačiam individui $i$ ankstensiu laikotarpiu $t - 1$.

Paprastumo dėlei tariama, jog nestebimi indvidų efektai $\mu_i$ nekoreliuoja su aiškinančiaisiais kintamaisiais $X_j$.

I dalis: duomenų simuliacija¶

  1. Parašykite funkciją gener_panel_data(), kuri sugeneruotų kintamuosius pagal nurodytas sąlygas:

gener_panel_data <- function() {
  N <- 40
  T <- 15
  burnin <- 500
  n <- T + burnin # stebiniai
  gamma <- 0.27
  beta_1 <- 0.73
  beta_2 <- 0.3
  zeta <- matrix(rpois(N * n, lambda = 10), nrow = n)
  eps <- matrix(rnorm(N * n, mean = 0, sd = 0.8), nrow = n)
  
  Y <- eps
  X_1 <- eps
  X_2 <- matrix(rchisq(N * n, df = 6), nrow = n)
  mu <- 1 * (1 - 0.27) * (mean(X_1[1, ]) - mean(X_1) + 1)
  
  for (i in 2:n) {
    X_1[i, ] <- 4 + 0.8 * X_1[i - 1, ] + zeta[i, ]
    mu[i] <- 1 * (1 - 0.27) * (mean(X_1[i, ]) - mean(X_1) + 1)
    Y[i, ] <- gamma * Y[i - 1, ] + beta_1 * X_1[i, ] + beta_2 * X_2[i, ] + mu[i] + eps[i, ]
  }
  
  Y <- Y[(burnin + 1):n, ]
  X_1 <- X_1[(burnin + 1):n, ]
  X_2 <- X_2[(burnin + 1):n, ]
  
  id <- rep(1:N, each = T)
  time <- rep((burnin + 1):n, N)
  
  df <- data.frame(cbind("time" = time, 
                         "id" = id, 
                         "Y" = as.vector(Y), 
                         "X1" = as.vector(X_1), 
                         "X2" = as.vector(X_2)))  
  
  return(df)
}
